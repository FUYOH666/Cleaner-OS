## Decision process

- **main** is protected in spirit: changes via PR with passing CI
- User-facing behavior changes require a CHANGELOG entry
- Security-sensitive defaults (dry-run, risk tiers) need explicit discussion before change

## Releases

- Semantic versioning from v1.0.0
- Tag format: `vMAJOR.MINOR.PATCH`
- Release notes sourced from [CHANGELOG.md](CHANGELOG.md)
